# GEANT4 Clinical Linear Accelerator Simulation 

A generic platform for clinical linear accelerator simulation using GEANT4. 

# Dependencies
* GEANT4
* CADMesh
* G4VoxelData
* Python, numpy
* PyUblas

# Installation
To compile the Python module:

    $> cd g4/
    $> mkdir build
    $> cd build/
    $> cmake ..
    $> make
